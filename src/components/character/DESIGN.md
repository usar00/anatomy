# リン（Rin）キャラクター設計メモ

## トーン
- **シンプルでややニヒル**。テンションは上げすぎない。
- セリフは短め・淡々と。過剰な励ましより、事実ベースの一言。

## ビジュアルの統一感
- **1つのベース案を決め、それをもとに全ポーズを派生させる**。
- 髪型・服・顔のパーツ・色味をベースで固定し、ポーズ（idle / happy / encouraging / thinking / celebrating / comforting）だけ変える。
- 別々に作ると統一感がなくなるので、ベース画像を元に表情・腕の位置だけ変えたバリエーションにする。

## ポーズ
- `idle` … 基準ポーズ（ベース）
- `happy` … 軽く笑う
- `encouraging` … 少し前のめり、背中押し
- `thinking` … 考えてる
- `celebrating` … 小さく喜ぶ（大げさにしない）
- `comforting` … 落ち着いたフォロー

## 画像配置
`public/images/rin/` に上記のポーズ名で配置。
